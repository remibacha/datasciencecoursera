# Welcome to the Code Book

## Fields
* subject - The person 's ID
* activity - The label of the activity performed when the corresponding measurements were taken
    
## Activity Labels

 *    WALKING (value 1)
 *    WALKING_UPSTAIRS (value 2)
 *    WALKING_DOWNSTAIRS (value 3)
 *    SITTING (value 4)
 *    STANDING (value 5)
 *    LAYING (value 6)
